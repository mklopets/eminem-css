How to use eminem.css

First include the main eminem.css stylesheet from the ./dist/css/ directory. Use the white_wrapper class in your HTML to mark an eminem element.

Alternatively, you can use the data-white-wrapper HTML attribute for that purpose. Should you prefer that approach, you will need to set the data-white-wrapper attribute to equal the string 'eminem'. Other attributes such as data-red-wrapper are not yet supported. Please see .
